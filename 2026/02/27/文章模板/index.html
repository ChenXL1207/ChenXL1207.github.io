📝 如何使用本模板

  1. 复制本文件：cp 文章模板.md source/_posts/你的文章标题.md
  2. 修改 FrontMatter（上面的—之间的内容）
  3. 删除这些说明，开始写你的文章
  4. 保存文件，刷新浏览器预览

FrontMatter 说明

字段 说明 必填 示例
title 文章标题 ✅ 我的第一篇技术文章
date 发布时间 ✅ 2026-02-27 12:30:00
tags 标签数组 ❌ [技术, 学习]
categories 分类 ❌ 技术心得
excerpt 首页摘要 ❌ 本文介绍…

💡 写作建议

  • 📌 标题清晰：能准确反映文章内容
  • 📝 内容充实：至少500字比较合适
  • 🔗 链接完整：外链最好使用https
  • 🏷️ 标签合理：3-5个标签最佳
  • 📅 时间准确：使用正确的发布时间
  • 🎯 分类单一：仅使用一个主分类

🎯 发布检查清单

发布前请检查：

  • 标题清晰准确
  • 内容完整无误
  • 代码高亮正确
  • 链接有效
  • 标签和分类合适
  • 图片能正常显示
  • 日期和时间正确

🚀 快速发布流程

  1. 在 source/_posts/ 创建 你的标题.md
  2. 复制本模板内容，修改FrontMatter
  3. 撰写文章内容
  4. 保存文件
  5. 运行 npm run server 查看预览
  6. 确认无误后，npm run build && npm run deploy 发布
